# Daniel Tools - Suite de Utilidades

Herramienta PowerShell con interfaz gráfica para administración de sistemas y bases de datos.

## Características
- Interfaz gráfica con Windows Forms
- Conexión a SQL Server
- Herramientas de instalación
- Gestión de impresoras
- Backup de bases de datos

## Requisitos
- Windows 10/11 o Windows Server 2016+
- PowerShell 5.0 o superior
- .NET Framework 4.5+

## Instalación
```powershell
# Clonar repositorio
git clone https://github.com/water0ff/dztools.git
cd dztools

# Ejecutar herramienta
.\src\tools.ps1
